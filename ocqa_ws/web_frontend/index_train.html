Tips

Write your question and press <TAB>. The appropriate fields below will be filled. If you think any of the fields is wrong, put the correct information, by copying from the question, and press the "Send" button, or type <ENTER>. Not all questions will have all the fields. Some of them may be blank.

For example: in the question "Is there any pizza restaurant near JAIST?"
The fields will be:

  • Question type: Place/Organization place or Existential
  • Question expression: Is there
  • Quantifier: any
  • Topic: pizza restaurant
  • Condition: near JAIST

Several other examples can be found in THIS file, but you don't need to follow the types declared there. If you think the question is about a place, select the place option independently of the expression used in the question.
